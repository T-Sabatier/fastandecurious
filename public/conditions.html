2. Le Jeu

  • Snap Tap est un jeu de cartes multijoueur à jouer entre amis, sur le même réseau de partie.
  • Le Jeu de base est gratuit. Aucun compte n'est nécessaire pour jouer.
  • Certains contenus (dont le « Mode Apéro », jeu à boire) sont réservés à un public majeur (18 ans et plus). Le Mode Apéro se joue avec la boisson de votre choix : une consommation sans alcool est préférable. Les règles à boire affichées sont de simples suggestions, sans aucune obligation de consommer de l'alcool. L'alcool est dangereux pour la santé, à consommer avec modération.
